# Mobile-Sales-Analytics-Using-Power-BI
This project features two interactive Power BI dashboards analyzing mobile sales, including Motorola. It provides brand-, model-, and region-wise analysis, forecasts future sales, detects anomalies, offers filters/KPIs/What-If simulations, and has future scope for AI/ML and Azure integration.
