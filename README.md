# 📊 Mobile Sales Data Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an **interactive Power BI-based data analytics solution** for analyzing mobile phone sales data across brands such as **Motorola and others**.  
The dashboards provide actionable insights into **sales performance, customer behavior, regional trends, and time-based patterns** using modern business intelligence techniques.

This project demonstrates **end-to-end data analytics**, including data preparation, modeling, visualization, and insight generation.

---

## 🧠 Analytics Performed
- **Descriptive Analytics** – Overall sales performance and KPIs
- **Diagnostic Analytics** – Performance dips, anomalies, and comparisons
- **Predictive Analytics** – Sales trend forecasting using time-based analysis
- **Prescriptive Analytics** – What-If analysis for decision support

---

## 🖥️ Dashboards Included

### 🔹 Dashboard 1: Mobile Sales Overview
Key insights provided:
- Total Sales, Quantity Sold, Transactions
- Brand-wise and Model-wise sales analysis
- City-wise and Region-wise sales distribution
- Payment method analysis
- Customer rating insights
